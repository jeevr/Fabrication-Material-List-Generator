# Fabrication Material List Generator

## Overview
**Fabrication Material List Generator (PyAutoFab)** is an automated fabrication material list generator that streamlines the process of creating and managing material requirements for fabrication projects.

## Features
- Automated material list generation
- Project-based organization
- Material tracking and management
- Export capabilities

## Installation
```bash
pip install pyautofab
```

## Usage
```python
from pyautofab import MaterialListGenerator

generator = MaterialListGenerator()
# Add your project details and materials
```

## Requirements
- Python 3.7+
- See `requirements.txt` for dependencies
